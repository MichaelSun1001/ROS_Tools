# csv2pcd
csv &amp; pcd convert each other
v1版本是进行单个文件转换，v2是整个文件夹转换，注意替换文件路径和文件夹路径即可

此脚本文件均是ubuntu系统下运行，windows未使用
使用时请用命令：
sudo python3 xxx.py
